Mengenal Sematic Elemen di HTML5

  • Article

    • Ketika kita ingin membuat layout untuk menampilkan block artikel yang ada di website kita, tag article bisa kita gunakan dengan elemen header. Karena tag article memiliki type block, artinya element ini memiliki lebar sesuai lebar browser jika kita tidak menentukan lebarnya.

  • Aside

    • Element ini bisa kita gunakan untuk melengkapi layout utama. Misalnya kita memiliki halaman blog maka kita bisa menambahkan aside di bagian bawah paragraf. Karena aside memiliki type/sifat block maka lebarnya sesuai page jika kita tidak menentukan lebarnya.
